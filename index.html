HTML Tags
Tag Name Description Requires Closing Tag
<Tag Name> Opening Tag Must be used at the beginning of all HTML tags. N/A
</Tag Name> Closing Tag Used to end or close an HTML element. When a section of HTML code has an opening tag, contents, and is properly closed it is called an HTML Element. Not all HTML tags need a closing tag, but must have an opening tag. N/A
<html> HTML All HTML code must be placed inside of an HTML element, in order to be processed properly.
<head> Head Holds Meta Data for HTML file, such as links to CSS files, the title to be displayed in the browser tab, and other resources. Most of this will not necessarily be displyed on the rendered page, as much as it will enhance it.
<body> Body This contains all the elements that will be displayed on the page. If an element is placed outside of this tag it may still be visible, but this should be avoided.
<h1> H1 Used for main headings. This tag can be used more than once if logical to do so, such as when sperated by semantic html. However, this should generally be avoided as h2-h6 tags can be used just as well in most cases.
<h2> - <h6> H2-H6 These are used for subheadings. The number following 'h' denotes priority, or contextual hierarchy. These should not be used for their default styling properties as this is better left to CSS. Rather, they should be used for semantics.
<p> Paragragh This tag is used to display and format text. While most HTML tags can do this, it should still be used semantically.
<span> Span This is used to encapsulate a section of text, usually for the purposes of styling it seperate from the rest of the surrounding text.
<em> Emphasize While this tag will italicize text by default in most cases, it should not be thought of a an italicizer. Rather it is used to convey the idea that there should be extra notice given to the affected text. There is a tag to italicize (<i>), but it should be avoided.
<strong> Strong This is similar to the Emphasize tag, but with more importance or urgency. The default styling will bold the affected text. There is a bold tag (<b>), but this should also be avoided.
<a> Anchor This tag is used to anchor a link to the page where the element is placed.
<br> Line Break This creates a line break. Whatever element comes after will be placed on a new line.
<img> Image This tag is used to add images to the page. A viable url to an image on the web, or relative path to a local file is needed.
<!--Text Here--> Comment The text inside of this element will not be displayed on the page. It is used the convey helpful information for anyone looking at the source code.
CSS
Property Description Value
color This sets the font color Color name, HEX, RGB, HSL
background-color This sets the background color of the selected element. Color name, HEX, RGB, HSL
font-size This sets the size of the font based on the chosen unit of measurment. px, pt, em, rem, %
font-family This sets the typeface of the text. Fonts chosen must be preinstalled or linked in order to be able to be displayed. Back-up fonts can be selected and chained with a comma. The browser will iterate through the list until an acceptable font is chosen. name of font
